## 项目简介

在太原理工大学（TYUT）计算机科学与技术学院的综合测评与奖学金评定工作中，传统的手工操作流程涉及大量数据比对、排名计算和资格审核，过程繁琐且易出错。

**CCST & CDS** 是一款基于 Python 开发的**综测奖学金自动化评定系统**，旨在将原本耗时的手工评定流程转化为「导入 - 计算 - 导出」的三步自动化操作。系统集成了荣誉奖评定、奖学金分级评定、同分排名处理、资格门槛校验等完整业务逻辑，最终输出可直接用于公示的 Excel 结果文件。

> 核心价值：将原本需要数小时的手工计算缩减至数秒内完成，消除人工核对的误差风险。

---

## 功能特性

### 1. 三步式评定流程
导入综测明细、荣誉名额、奖学金名额三张表，一键启动自动评定，输出荣誉名单、奖学金名单、汇总全表。

### 2. 智能表头识别
自动检测 Excel 表头行位置，兼容不同模板的列名差异（如"美育"自动映射为"文化艺术"），减少格式调整工作量。

### 3. 完整评定规则
内置劳动实践门槛(>2.0)、身心素质资格线(≥27.0)、学业成绩百分比排名、荣誉奖数量门槛等全部评定规则。

### 4. 同分智能排序
支持以"综测成绩"或"学业成绩"作为同分时的辅助排序依据，灵活适配不同院系的评定需求。

### 5. 数据兼容性强
支持 `.xlsx` / `.xls` / `.csv` 多种格式导入，自动处理 BOM、全角空格、文本数字混合等常见数据质量问题。

### 6. 一键打包可执行
通过 PyInstaller 打包为独立 `.exe` 程序，无需安装 Python 环境即可运行，方便分发到各班委使用。

## 评定规则

### 资格门槛

- **身心素质评分** 须 ≥ 27.0，低于此线者取消全部评奖评优资格
- **劳动实践** 项评分须 > 2.0 方可参评对应荣誉奖
- 其他荣誉奖项评分须 > 0

### 荣誉奖项（10 类）

| 类别 | 说明 |
|------|------|
| 学业优秀 | 基于学业成绩排名 |
| 社会工作 | 学生工作、社团工作 |
| 学术研究 | 科研、论文、课题 |
| 创新创业 | 创新创业大赛、科创 |
| 公益服务 | 志愿服务、义工 |
| 对外交流 | 境外交流、交流实践 |
| 体育活动 | 体育类、体育竞赛 |
| 文化艺术 | 文艺、美育、艺术实践 |
| 劳动实践 | 劳动教育、社会实践 |
| 自强自立 | 困难生自强自立 |

### 奖学金等级与门槛

| 奖学金等级 | 学业成绩排名要求 | 荣誉奖数量门槛 | 奖励金额 |
|:----------:|:--------------:|:------------:|:--------:|
| 一等奖学金 | 前 20% | ≥ 3 项 | **1800 元** |
| 二等奖学金 | 前 50% | ≥ 2 项 | **1200 元** |
| 三等奖学金 | 前 60% | ≥ 1 项 | **600 元** |

## 快速开始

### 方式一：直接运行可执行文件

双击 `CCST&CDS.exe` 即可启动，无需安装任何依赖。

### 方式二：从源码运行

```bash
# 1. 克隆项目
git clone https://github.com/your-username/CCST-CDS.git
cd CCST-CDS

# 2. 安装依赖
pip install pandas numpy openpyxl

# 3. 启动系统
python "CCST&CDS.py"
```

### 方式三：打包为可执行文件

```bash
pyinstaller --onefile --windowed --icon="CCST&CDS.ico" --name="CCST&CDS" "CCST&CDS.py"
```

## 使用指南

### 模板文件

项目提供了以下 Excel 模板文件，可参考格式准备数据：

- **计算机综测总表.xlsx** — 综合测评明细数据模板
- **奖学金名额分配-模板.xlsx** — 各等级奖学金名额配置模板
- **荣誉奖名额分配-模板.xlsx** — 各类荣誉奖项名额配置模板

### 操作步骤

1. 点击「导入综测明细」加载学生综测数据
2. 点击「导入荣誉名额」加载各类荣誉奖项名额
3. 点击「导入奖金名额」加载奖学金各等级名额
4. 根据需要选择同分排序参考依据（综测成绩 / 学业成绩）
5. 点击「开始自动评定」运行计算
6. 评定完成后，分别导出荣誉名单、奖学金名单和汇总全表
